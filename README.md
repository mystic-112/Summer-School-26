The IoT, Robotics & Drones Internship Program is a structured learning and hands-on training initiative designed to build a strong foundation in embedded systems, IoT, robotics, and autonomous technologies.

The program combines theoretical concepts with practical implementation using real hardware components, embedded programming, wireless communication, automation workflows, and intelligent control systems.

Participants gain exposure to modern industrial technologies including IoT platforms, robotics control systems, drone technologies, and smart automation solutions.

# Final Project

## BLE-Based Smart Workforce Presence and Availability Monitoring System Using ESP32 and IoT Analytics

### Project Overview

This project utilizes Bluetooth Low Energy (BLE) technology and ESP32 microcontrollers to monitor workforce presence and availability in indoor environments.

BLE-enabled devices periodically broadcast advertisement packets which are detected by ESP32 scanning nodes. Using RSSI-based proximity estimation, the system identifies presence patterns and occupancy information in real time.

Collected data can be transmitted to cloud platforms for monitoring, analytics, and visualization.

### Key Features

* BLE-based indoor presence detection
* Real-time workforce availability monitoring
* Low power operation using BLE technology
* MQTT-enabled IoT analytics integration
* Scalable deployment architecture
* Occupancy and utilization analysis

### Technologies Used

* ESP32
* Bluetooth Low Energy (BLE)
* Wi-Fi
* MQTT
* Arduino IDE
* IoT Analytics Platforms

### Applications

* Smart Offices
* Industrial Facilities
* Educational Campuses
* Research Laboratories
* Smart Buildings
